TECHNICAL DETAILS:
================================================================================

Database Column:
- Table: EmployeeMaster
- Column: EmploymentType
- Type: NVARCHAR(50) NULL
- Constraint: CHECK (EmploymentType IN ('Permanent', 'Contract', 'Guest') OR EmploymentType IS NULL)

Stored Procedure:
- Name: Proc_Executive_set
- New Parameter: @EmploymentType NVARCHAR(50) = NULL
- Position: After @Experience, before @CreatedBy

Form Field:
- ID: employmentType
- Name: employmentType
- Type: select (dropdown)
- Required: Yes
- Options: Permanent, Contract, Guest

# Indian Standard Student ID Card Templates

## Overview
Creating 14 modern, professional Student ID Card templates following Indian educational institution standards.

## Template Specifications

### Standard Information Fields:
- Student Name
- Student ID/Code
- Class & Section
- Roll Number
- Date of Birth
- Contact Number
- School Name & Logo
- Student Photo

### Design Standards:
- **Horizontal Cards**: 450px × 180px (Credit card ratio)
- **Vertical Cards**: 240px × 410px (Standard ID card ratio)
- **Professional color schemes**
- **Clear typography**
- **FontAwesome icons**
- **Proper spacing and alignment**

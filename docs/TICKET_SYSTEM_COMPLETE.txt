INSTALLATION INSTRUCTIONS
================================================================================

STEP 1: DATABASE INSTALLATION
────────────────────────────────────────────────────────────────────────────
Open SQL Server Management Studio and execute:

    :r database\ticket_system\INSTALL_TICKET_SYSTEM.sql

This will:
- Create all 11 tables with indexes
- Insert default data (categories, priorities, statuses, SLAs)
- Create all 6 stored procedures
- Add Support Executive role to ProfileMaster
- Add menu integration
- Verify installation

STEP 2: DJANGO INTEGRATION
────────────────────────────────────────────────────────────────────────────
Add to core/urls.py:

    from django.urls import path, include
    
    urlpatterns = [
        # ... existing patterns ...
        path('tickets/', include('core.ticket_system.urls')),
    ]

STEP 3: RESTART DJANGO
────────────────────────────────────────────────────────────────────────────
    python manage.py runserver

STEP 4: VERIFICATION
────────────────────────────────────────────────────────────────────────────
Test queries:

    -- Verify tables
    SELECT COUNT(*) FROM TicketMaster;
    SELECT COUNT(*) FROM TicketCategory;
    
    -- Verify procedures
    SELECT name FROM sys.procedures WHERE name LIKE 'Proc_Ticket%';
    
    -- Verify ProfileMaster
    SELECT * FROM ProfileMaster WHERE ProfileID = 5;
    
    -- Verify menus
    SELECT * FROM MenuMaster WHERE MenuName LIKE '%Ticket%';

STEP 5: TEST API
────────────────────────────────────────────────────────────────────────────
    curl http://localhost:8000/tickets/categories/
    curl http://localhost:8000/tickets/priorities/
    curl http://localhost:8000/tickets/dashboard/

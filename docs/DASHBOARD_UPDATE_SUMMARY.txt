================================================================================
CHANGES MADE
================================================================================

1. STORED PROCEDURE (Proc_Dashboard_Students_Get.sql)
   NEW PARAMETERS:
   - @FromDate (DATE) - Filter students created from this date
   - @ToDate (DATE) - Filter students created until this date
   - @ShowActiveOnly (BIT) - Default: 1 (show only active students)
   
   UPDATED LOGIC:
   - Added date range filtering on Student.CreatedAt column
   - Added active-only filter (IsActive = 1)
   - Applied filters to both summary and class breakdown queries

2. BACKEND (core/views.py)
   
   dashboard_view():
   - Changed to show active students by default (@ShowActiveOnly = 1)
   - Total students now shows active count instead of all students
   
   api_dashboard_students():
   - Added from_date parameter
   - Added to_date parameter
   - Added show_active_only parameter (default: '1')
   - Passes all parameters to stored procedure

3. FRONTEND (core/templates/core/dashboard.html)
   
   FILTER SECTION:
   - Added "From Date" input field (type="date")
   - Added "To Date" input field (type="date")
   - Added "Reset" button next to "Apply" button
   - Wrapped entire filter section in profile check: {% if user.profile_id == 2 or user.profile_id == 3 %}
   
   DASHBOARD CARDS:
   - Changed "Total Students" to "Active Students"
   - Wrapped student cards in profile check: {% if user.profile_id == 2 or user.profile_id == 3 %}
   - Only School Admin and Teacher can see student statistics cards
   - Other profiles see only Teachers and Fees cards
   
   JAVASCRIPT:
   - Added date filter handling in applyFilters()
   - Added show_active_only parameter (always '1')
   - Added resetFilters() function to clear all filters
   - Reset button clears all inputs and triggers apply

================================================================================
DEFAULT BEHAVIOR
================================================================================

ON PAGE LOAD:
- Shows ONLY active students (IsActive = 1)
- No date filters applied
- No class/section/gender/category filters applied
- Shows all active students for the school

AFTER APPLYING FILTERS:
- Still shows only active students
- Applies date range if specified
- Applies other filters if selected
- Updates cards in real-time

AFTER RESET:
- Clears all filter inputs
- Automatically triggers "Apply Filters"
- Returns to default state (all active students)

================================================================================
DATE FILTER USAGE
================================================================================

EXAMPLES:

1. Students created in January 2024:
   From Date: 2024-01-01
   To Date: 2024-01-31
   Click "Apply"

2. Students created after June 1, 2024:
   From Date: 2024-06-01
   To Date: (leave empty)
   Click "Apply"

3. Students created before December 31, 2023:
   From Date: (leave empty)
   To Date: 2023-12-31
   Click "Apply"

4. All active students (no date filter):
   From Date: (leave empty)
   To Date: (leave empty)
   Click "Apply" or "Reset"

📊 TEMPLATE VARIABLES
═══════════════════════════════════════════════════════════════════════════════

School Information:
  • school_name          - School name
  • school_logo          - School logo (base64/URL)
  • school_address       - Full address
  • school_phone         - Contact number
  • school_email         - Email address

Receipt Information:
  • receipt_no           - Receipt number
  • date_of_submission   - Payment date
  • fees_month           - Fee period/month

Student Information:
  • full_name            - Student full name
  • student_code         - Registration number
  • class_name           - Class name
  • section_name         - Section name

Fee Details:
  • fee_breakdown        - List of fee items
    ├─ name              - Fee item name
    └─ user_enter_amount - Fee amount
  • total_amount         - Total fee amount
  • paid_amount          - Amount paid
  • remaining_amount     - Balance remaining


🚀 HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

1. Navigate to Template Management
   URL: /template-management/

2. Find "Fee Receipt" Card
   Icon: 💵

3. Select Template
   Choose from 5 options in dropdown

4. Preview Template
   Click "Preview" button to see sample

5. Apply Template
   Click "Apply" to save selection

6. Template is Now Active
   Will be used for all fee receipts


💡 INTEGRATION EXAMPLE
═══════════════════════════════════════════════════════════════════════════════

# Get selected template for school
def get_fee_receipt_template(school_id):
    with connection.cursor() as cursor:
        cursor.execute(
            "EXEC Proc_Template_Preference_Get @SchoolID = %s", 
            [school_id]
        )
        templates = {}
        for row in cursor.fetchall():
            templates[row[0]] = row[1]
    
    return templates.get(
        'FeeReceipt', 
        'core/document_templates/fee_receipt/fee_receipt_template1.html'
    )

# Use in view
def print_fee_receipt(request, receipt_id):
    school_id = request.session.get('SchoolID')
    template = get_fee_receipt_template(school_id)
    
    context = {
        'receipt_no': 'FEE-001',
        'full_name': 'John Doe',
        'fee_breakdown': [...],
        # ... other data
    }
    
    return render(request, template, context)

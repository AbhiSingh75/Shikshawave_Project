## Template 1: Purple Gradient
**Style**: Modern, Vibrant  
**Color Scheme**: Purple (#667eea) to Violet (#764ba2)  
**Best For**: Modern schools, tech-focused institutions

### Features:

- Rounded corners (15px)

- Alternating row colors (#f8f9ff)
- Gradient header

### Visual Characteristics:
- **Header**: Purple gradient with white text
